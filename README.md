# Descrição do Projeto
<p>O conversor de moedas é uma API feito para rodar localmente na sua maquina. A finalidade da API é converter mais 150 tipos de moedas mundiais que abrange 200 paises.</p>

# Funcionalidades e Demonstração da Aplicação
# :hammer: Funcionalidades do projeto
- `verificar valor da moeda`: A aplicação verifica o valor que determinada moeda possui quando convertida para outra.
- `Converter moeda para outra moeda passando um valor`: A aplicação também converte um valor passado junto com o tipo da moeda.
